Quick Start

Installation

bash
$ git clone https://github.com/maxiarat1/ai-image-captioner.git
$ cd ai-image-captioner
$ ./setup-captioner-gpu.sh
$ conda activate captioner-gpu
$ cd backend && python app.py

Requirements

  • NVIDIA GPU with CUDA 12.8+ (RTX 20/30/40/50 series)
  • Python 3.12
  • 4GB+ VRAM minimum (20GB+ recommended for R-4B)

API Reference

POST /generate

Generate a caption for a single image.

python
import requests
with open('image.jpg', 'rb') as f:
response = requests.post(
'http://localhost:5000/generate',
files={'image': f},
data={'model': 'wdvit', 'prompt': 'Describe this image'}
)
print(response.json()['caption'])

Troubleshooting

Out of Memory Error

Solutions:

  • Use lightweight models (BLIP, WD-ViT) instead of larger models
  • Switch to lower precision (8-bit or 4-bit) for supported models
  • Close other GPU-intensive applications

CUDA Version Mismatch

Ensure you're using the correct build:

  • RTX 20/30/40/50 series: python312-cuda128

Key Features
